### Langkah-langkah  
1. **Clone Repository**  
   ```bash
   git clone https://github.com/Informatics-ITS/ta-chrstnkevin.git
   ```
2. **Instalasi Dependensi**
   ```bash
   cd ta-chrstnkevin
   npm i
   ```
4. **Jalankan Aplikasi**
   ```bash
   npm run dev
   ```
5. Buka browser dan kunjungi: `http://localhost:3000/component`
